## 📋 [시작 프롬프트 1: 프로젝트 설정 & 디자인 시스템] (~1,500자)

## 🎯 프로젝트 개요

**프로젝트명:** WOORIDO (우리두)
**태그라인:** "우리를 위한 계모임 커뮤니티, 우리두"
**핵심 가치:** "기술이 보장하는 가장 안전한 취향 기반 커뮤니티"
**테마:** 고객 수입/지출 데이터 분석 기반 지역 커뮤니티 자금 솔루션

**핵심 차별화 (3계층 신뢰 아키텍처):**
```
Layer 1: 💰 재정 분석 (Django + pandas로 고객 재정 데이터 분석)
         ↓
Layer 2: ⭐ 신뢰 트라이포드
         - 🔒 보증금 잠금 (선불 보증금)
         - 📊 장부 투명성 (Recharts 시각화)
         - 🗳️ 합의 지불 (투표 기반 승인)
         ↓
Layer 3: 🔍 취향 검색 (Elasticsearch 태그/텍스트 검색)
```

**기술 스택:**
- Frontend: React + TypeScript + Radix UI + Tailwind CSS
- Charts: Recharts (Line, Pie, Bar)
- Responsive: 모바일 우선 (< 768px 모바일, ≥ 768px 데스크톱)

---

## 📱 반응형 디자인 전략

**브레이크포인트:**
- 모바일: < 768px (기본 스타일, 320px ~ 767px)
- 데스크톱: ≥ 768px

**디자인 원칙:**
- 모바일: 세로 스택 레이아웃, 하단 탭 바, 전체화면 카드
- 데스크톱: 그리드 레이아웃, 사이드바, 다단 컬럼

**색상 테마:**
- **기본: 코랄 오렌지 #E85D3E** (로고의 WOORIDO 브랜드 색상)
- 보조: 검정 #000000 (로고 텍스트 색상)
- 강조: 따뜻한 오렌지/코랄 변형 (#FF6B4A, #FF8566)
- 배경: 흰색/밝은 회색 (#FFFFFF, #F5F5F5)
- 성공: 녹색 (#10B981)
- 경고: 노랑 (#F59E0B)
- 오류: 빨강 (#EF4444)

**타이포그래피:**
- 폰트: Pretendard 또는 Noto Sans KR
- 간격: 8px 기준 (8, 16, 24, 32)
